# Checklist de Preparación para Integraciones y Fase 2

## Datos y gobierno
- [ ] Catálogo definitivo de niveles/cursos validado
- [ ] Reglas de negocio de estatus académico validadas
- [ ] Política de deduplicación de alumnos/representantes definida
- [ ] Diccionario de datos maestro aprobado

## Integraciones externas
- [ ] APIs de BioAPP documentadas (endpoints, auth, límites)
- [ ] APIs de Kommo documentadas (objetos, webhooks, auth)
- [ ] Ambiente sandbox habilitado para cada integración
- [ ] Matriz de campos de sincronización (origen/destino) aprobada

## Mensajería y notificaciones
- [ ] Proveedor WhatsApp definido (Cloud API/agregador)
- [ ] Plantillas aprobadas y política de envíos definida
- [ ] Estrategia de fallback (email/SMS) definida

## Aula virtual y contenido
- [ ] Proveedor de videoconferencia seleccionado
- [ ] Reglas para tareas, entregas y calificación aprobadas
- [ ] Política de almacenamiento multimedia (S3 u otro) definida

## Infraestructura
- [ ] Ambiente staging y producción con backups
- [ ] Monitoreo y alertas técnicas configuradas
- [ ] Política de retención de logs establecida

## Producto
- [ ] Backlog Fase 2 priorizado por valor
- [ ] Criterios de aceptación por módulo aprobados
- [ ] Plan de UAT Fase 2 definido
